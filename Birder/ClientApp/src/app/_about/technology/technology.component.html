The Technology Stack

Here is a summary of the technology used by Birder:

Client

  • Angular 121 and TypeScript single-page application
  • Bootstrap and Angular Material style and components libraries

Server

  • ASP.NET 52 and C# API
  • Azure SQL Database

Deployment

  • Hosted on the Azure cloud platform in Docker containers
1 Since 15th May 2021. Formerly Angular 6 - 11.
2 Since 13th November 2020. Previously .NET Core 2.2, 3.0, and 3.1.
